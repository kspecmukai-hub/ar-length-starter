# AR Length Check Starter

## これは何？
- スマホのカメラ映像に **丈感シルエット** を重ねて確認できる簡易「AR風」ページ（`index.html`）。
- 3DモデルをAR表示するための `<model-viewer>` サンプル（`ar-detail.html`）。

## 使い方（ローカル確認）
1. このフォルダをPCにダウンロード。
2. 任意の静的ホスティング（GitHub Pages / Netlify / Vercel など）にアップロード。
3. 公開URLをスマホで開く。PCで開くと右下にQRが出ます。

## MakeShop への組み込み
- 商品ページにボタンまたはQRコードを配置し、`index.html` もしくは `ar-detail.html` の公開URLへリンクします。
